Nuxt x Component x Import 的小事

TL;DR

NuxtJs

NuxtJs 是一款基於 VueJs 的前端框架，延伸實作了 SSR (Server-Side Render) 跟一些幫助開發跟理解的語法。

Component

當我們發現好幾頁都有相同的部分時，就可以把它們提取出來寫成 元件 (Component)，減少重複的程式碼。

根據 Nuxt 文件的描述，
將 vue 程式碼放在 components 資料夾中，就能直接被其他檔案所引用。
