HTML Tag case-insensitive

Posted on 2020-12-31 In Web

HTML tag 竟然只有小寫！

分享了上一篇文章 Nuxt x Component x Import 的小事 之後，隔天我才知道，這是誤會中的誤會，朋友跟我說這不是這個原因。

真正的原因，是 大小寫！

HTML tag

前情提要，上一篇文章中，我在 page 中要 import component，嘗試的結論是在 layout 中可以，但 page 中不行，這其實是個錯誤的結論。

實際上是因為 HTML tag 本身是不分大小寫的 (case-insensitive)，即使有大小寫也只會抓小寫的檔案，

tag names are case-insensitive.[1]

而我的 Component 命名為 shareButton.vue，HTML 中試圖使用時，即使我們寫了 <shareButton />，他依舊認為這是不同的東西，

如果把檔名及 HTML tag 都改成小寫之後，就不需要顯式的去 Import 這個 component 了，可喜可賀。

[1]: HTML Standard
